# Advance a task

Read the repository-root `AGENTS.md` in full, then follow its **How to advance a task one stage** procedure exactly. `AGENTS.md` is authoritative for stage requirements, sign-off rules, model identification, file movement, and reporting.

The invocation requires a target task path. If the user did not name one, ask which file to advance rather than choosing a task yourself.

Keep this file as a thin discovery adapter. Do not copy the transition procedure here; update `AGENTS.md` when the shared workflow changes.
