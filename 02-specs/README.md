# 02-specs

Tasks here have technical depth: an architecture/approach with trade-offs, an `## Action Items` section (anything needing manual/external setup a coding agent can't do on its own — creating an account, registering an app in a browser console, obtaining an API key), and an `## Open Questions` section (decisions only a human can make).

Tasks arrive here from `01-ideas/` via the `advance-task` procedure, and move on to `03-ready/` the same way once the open questions are resolved and the approach is settled. See `/AGENTS.md` at the repo root for the full pipeline description.
